## Chat Interface

====

## Objective

Developed a chat interface that simulates real-time messaging.

---

## Description

This project is a simple chat application that allows users to send messages and receive simulated responses. It mimics real-time communication using JavaScript timers like `setTimeout` and `setInterval`.

The application dynamically updates the chat window as messages are sent and received.

---

## Features

- Send and display user messages  
- Simulate incoming messages using timers  
- Display timestamps for each message  
- Dynamic chat updates without page reload  
- Simple and interactive chat UI  

---

## Technologies Used

- HTML  
- CSS  
- JavaScript 

---

## How It Works

1. The user types a message and sends it.
2. The message is displayed instantly in the chat window.
3. A simulated reply is generated using `setTimeout` or `setInterval`. The reply is sent after a random delay between 3 and 7 seconds. During this time, a "typing..." indicator is shown.
4. The incoming message appears after the delay.
5. Each message includes a timestamp.
6. The chat window updates dynamically as the conversation continues.
