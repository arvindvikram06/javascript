## E-Commerce shopping cart

====

## Description

This is a simple e-commerce application built using modular JavaScript. Users can view products, check product details, and add items to a cart.

---

## Features

- View products (fetches from dummy api) 
- View product details  
- Add to cart  
- Increase/decrease quantity  
- Remove items from cart  
- Cart stored in localStorage  

---

## Technologies Used

- HTML  
- CSS  
- JavaScript

---

## How It Works

1. Products are fetched from an API.
2. Products are displayed on the homepage.
3. Users can view product details.
4. Items can be added to the cart.
5. Cart is stored in localStorage and updated dynamically.
