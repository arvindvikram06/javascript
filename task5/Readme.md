## Quiz App

====

## Description

This is a simple Quiz Application built using JavaScript. It allows users to answer multiple-choice questions and displays the final score at the end.

---

## Features

- Displays one question at a time  
- Multiple-choice options for each question  
- Tracks user answers  
- Calculates and shows final score  
- Simple and interactive user interface  
- uses localstorage to persist the user's progress

---

## Technologies Used

- HTML  
- CSS  
- JavaScript  

---

## How It Works

1. The app loads a set of questions.
2. Each question is displayed with multiple options.
3. The user selects an answer and moves to the next question.
4. After all questions are completed, the score is calculated.
5. The final result is displayed.
