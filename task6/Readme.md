## Drag and Drop List

====

## Description

This project is a simple web application that enables users to rearrange list items by dragging and dropping them. It uses the HTML5 Drag and Drop API to handle drag events and update the order of items dynamically.

---

## Features

- Drag and drop to reorder list items  
- Real-time DOM updates after dropping items  
- Visual feedback during drag operations  
- Smooth and interactive user experience  

---

## Technologies Used

- HTML  
- CSS  
- JavaScript
- HTML5 Drag and Drop API  

---

## How It Works

1. The list of items is displayed on the page.
2. The user clicks and drags an item.
3. Drag events are triggered using the HTML5 Drag and Drop API.
4. As the item is dragged, potential drop targets are highlighted.
5. When the item is dropped, the DOM is updated to reflect the new order.
