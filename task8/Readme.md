## Single Page Application (SPA)

====

## Objective

Build a basic Single Page Application that navigates between different views without reloading the page.

---

## Description

This project is a simple SPA that uses hash-based routing to switch between different views. Instead of loading new pages, the application dynamically updates the content based on the URL hash.

---

## Features

- Navigation without page reload  
- Hash-based routing  
- Dynamic content rendering    
---

## Technologies Used

- HTML  
- CSS  
- JavaScript

---

## How It Works

1. The application listens for changes in the URL hash using `window.onhashchange`.
2. When the hash changes, the app determines which view to display.
3. The corresponding content is dynamically loaded into the page.
